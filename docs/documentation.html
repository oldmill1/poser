Poser Documentation

Installation

  1. Clone the repository: `git clone https://github.com/ataxali/poser.git`
  2. Navigate to directory: `cd poser`
  3. Create virtual environment: `python3 -m venv venv`
  4. Activate environment: `source venv/bin/activate` (Linux/Mac) or `venv\Scripts\activate` (Windows)
  5. Install package: `pip install -e .`
  6. Set up API key: `export OPENAI_API_KEY='your-api-key-here'`

Basic Usage

  1. Check profile status: `poser`
  2. Generate text: `poser 'write a slack message about my new app deployment'`
  3. Add writing sample: `poser add-sample 'slack update' 'Hey team, quick update...'`
  4. List samples: `poser list-samples`
  5. Remove sample: `poser remove-sample sample_20250911_185212`
  6. Backup profile: `poser --backup`

How It Works

  1. **Add Writing Samples**: Provide examples of your writing style
  2. **AI Analysis**: Poser analyzes each sample to understand your patterns
  3. **Smart Matching**: When requesting text, it finds relevant samples
  4. **Style Generation**: Generates new text that matches your authentic voice
