Features

  1. Personal style analysis from writing samples
  2. Context-aware text generation
  3. Command-line interface for easy usage
  4. OpenAI GPT integration
  5. Sample management and backup system
  6. Cross-platform Python support

Quick Start

  1. Install: `pip install -e .`
  2. Setup: `export OPENAI_API_KEY='your-key'`
  3. Check status: `poser`
  4. Generate text: `poser 'write a slack message about my app'`
